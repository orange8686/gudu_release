# 咕嘟鸡 - 正式版发布仓库

这是 **咕嘟鸡** App 的官方分发与版本追踪仓库。

## 🚀 项目简介

**咕嘟鸡** 是一款基于 Flutter 开发的轻量化混合博客 App，旨在为用户提供极致流畅的 Typecho 博客阅读体验。

本仓库仅用于存放编译后的正式版安装包（APK）及更新日志。

## 📦 下载说明

你可以前往本仓库的 [Releases](https://github.com/orange8686/gudu_release/releases) 页面下载最新版本的 APK 安装包。

- **稳定版**: 经过完整测试的正式版本。
- **架构支持**: 目前主要提供 `arm64-v8a` 架构（适配绝大多数现代安卓手机）。

## 🛠️ 自动化发布原理

本项目采用现代化的 **CI/CD (持续集成与持续发布)** 流程：

1. **私有构建**: 核心源代码托管在私有仓库，确保开发环境的安全。
2. **自动化流水线**: 当私有仓库推送新的版本标签（如 `v1.0.3`）时，GitHub Actions 会自动触发编译。
3. **跨仓库分发**: 编译完成并经过正式签名的 APK 会自动同步到本公开仓库。
4. **加速分发**: 为了优化国内用户的下载体验，App 内部集成了镜像加速服务（如 `gh.felicity.ac.cn`），确保一键更新的顺畅。

## 📢 更新检测

如果你已经安装了咕嘟鸡，可以直接在 App 内的 **「设置 -> 关于 -> 检查更新」** 中直接获取最新版本的更新日志并一键升级。
